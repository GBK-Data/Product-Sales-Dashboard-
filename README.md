# Product Sales Dashboard (Power BI)

## Overview
This project is a sales performance dashboard built in Power BI. It analyzes key business metrics such as revenue, profit, orders, product categories, and regional performance.

## Tools
- Power BI

## Key Metrics
- Total Revenue
- Total Units Sold
- Profit (2011-2014)
- Product Performance
- Category Performance
- Regional Performance

## Key Insights
- Clear seasonality with December as the strongest revenue month
- Central region is the main driver of revenue and orders
- Technology is the best-performing category driven by high-value products like smartphones
- Profit shows a consistent upward trend over the years

## Learning Outcome
This project helped me understand how to move from raw data to simple business insights and improve my analytical thinking.
